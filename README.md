## 📌 Descripción

**GestiDomus** es una aplicación web tipo SaaS orientada a la gestión de propiedades en alquiler.

El objetivo del proyecto es centralizar en una única plataforma la administración de inmuebles, contratos, pagos mensuales, incidencias y documentación asociada, facilitando el control operativo y financiero para propietarios o gestores inmobiliarios.

Este proyecto ha sido desarrollado como una aplicación full-stack moderna, combinando una interfaz web clara con backend, base de datos, autenticación, almacenamiento de archivos y generación documental.

## ✨ Funcionalidades principales

- Gestión de inmuebles.
- Registro y administración de contratos.
- Control de pagos mensuales.
- Gestión de incidencias.
- Panel de control con métricas principales.
- Generación de recibos en PDF.
- Gestión documental.
- Almacenamiento de archivos.
- Integración con Supabase.
- Despliegue en Vercel.
- Interfaz moderna y responsive.
